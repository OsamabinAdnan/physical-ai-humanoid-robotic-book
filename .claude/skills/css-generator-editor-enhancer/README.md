# CSS Generator, Editor and Enhancer Skill

This skill provides comprehensive CSS generation, editing, and enhancement capabilities. It helps users create, modify, optimize, and enhance CSS code with modern features and best practices.

## Structure

```
css-generator-editor-enhancer/
├── SKILL.md                 # Main skill definition
├── README.md               # This file
├── reference.md            # CSS best practices reference
├── examples.md             # Usage examples
├── scripts/                # CSS manipulation scripts
│   ├── generate-css.js     # Generate CSS from specifications
│   └── css-validator.js    # Validate CSS syntax
├── assets/                 # CSS templates and assets
│   ├── common-components.css  # Common component templates
│   └── css-variables.css      # CSS variables template
└── templates/              # (Optional) Additional templates
```

## Features

### Scripts
- **generate-css.js**: Generate CSS based on design specifications
- **css-validator.js**: Validate CSS syntax and best practices

### Assets
- Predefined CSS templates for common components
- CSS variables template with consistent theming
- Utility classes similar to Tailwind CSS

### References
- Comprehensive CSS property reference
- Guide to modern CSS techniques
- CSS best practices documentation

## Usage

This skill should be triggered when users need:
- To generate CSS from scratch based on requirements
- To modify or edit existing CSS
- To enhance CSS with modern features
- To optimize CSS for performance
- To validate CSS syntax and best practices

## Examples of Use Cases

1. **CSS Generation**: "Create CSS for a responsive navigation bar"
2. **CSS Enhancement**: "Modernize this CSS with flexbox and variables"
3. **CSS Validation**: "Check if this CSS follows best practices"
4. **Component Creation**: "Generate a card component with hover effects"

## Best Practices Implemented

The skill follows modern CSS best practices including:
- CSS custom properties (variables) for maintainability
- Flexbox and grid for layouts
- Logical properties for internationalization
- Modern units (rem, em, fr, etc.)
- Accessibility considerations
- Performance optimization techniques
